### Server Components

The future of React with server components:

```jsx
// Server Component
async function BlogPosts() {
  const posts = await db.posts.findMany();
  return (
    <ul>
      {posts.map((post) => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
}
```
